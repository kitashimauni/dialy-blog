2泊3日で宮崎旅行に行きました。

中学の頃からの友人たち4人と一緒で、友人の親戚にはかなりお世話になりました。

せっかくなので訪問場所などの記録を残しておきます。

# 1日目
## 羽田空港から宮崎空港へ
行きは羽田から宮崎へ2時間弱のフライト。航空会社はJALです。国内線のFSCに乗ったのはかなり久しぶりな気がします。

トラブルもなく宮崎空港に到着。宮崎空港には航空大学校があるらしいです(写真の奥に映るのは航空大学校の格納庫)。

{% image "./PXL_20260320_115902919.MP.jpg", "宮崎空港" %}

## 昼食
昼食は『[えぷろん亭](https://maps.app.goo.gl/EdbzgqGwCBZm4pYs9)』というところでとりました。

海鮮丼セットを食べました(数量限定らしい)。チキン南蛮や蟹の出汁が効いた味噌汁もついてきて満足です。

{% image "./PXL_20260320_130155751.jpg", "昼食", "100vw" %}

海岸沿いの少し高いところにあるのでめちゃくちゃ景色がいいです。

{% image "./PXL_20260320_123750671.MP.jpg", "えぷろん亭からの景色", "100vw", "50%" %}

## サンメッセ日南
昼食をとって次に行ったのはサンメッセ日南です。

こちらも海沿いにある施設で、本家公認のモアイ像があります。

日本の修復チームがモアイ像(本物)を修復したお礼としてモアイを復刻することを許可されたようです。

{% image "./PXL_20260320_150751536.jpg", "モアイ像", "100vw", "100%" %}

モアイ像の後ろにはオーシャンビューが広がっていて、非常にきれいです。

都市部だと、海を見ると船がたくさん浮かんでいますが、ここでは船はほとんどいません。

かなり高低差があり、歩いて登ることもできますが、免許を持っている人がいればラウンドカーを運転して登ることもできます。

実際に運転して登りましたが、道が狭いのですれ違いが難しいです。

モアイ以外にもレストランやオブジェがあるので見どころはそれなりにあるのではなかろうか。

## 都井岬
正確には小松ヶ丘というらしい。

着いた頃には夕暮れ時、日が落ちかけていてかなりいい感じの雰囲気でした。景色もとてもよい。

{% image "./PXL_20260320_180153210.MP.jpg", "都井岬からの風景", "100vw", "100%" %}

また、ここには馬もいます。その代わりフンだらけなので要注意…

{% image "./PXL_20260320_172408612.jpg", "馬", "100vw", "100%" %}

## 夕食
温泉に入ったあと、近くの居酒屋で夕食をとりました。

余り特筆すべきことはないですが、本霧島を飲みました。宮崎限定らしいです。

黒霧島と飲み比べましたが、本霧島のほうが飲みやすかったです。

今回は買ってないですがお土産にいいかも。

# 2日目
## 日向岬・馬ヶ背
2日目初日は日向岬・馬ヶ背に行きました。

日向岬には細島灯台という灯台があります。しっかりメンテナンスされているのか外観がきれいです。

{% image "./PXL_20260321_124725000.jpg", "日向岬の灯台", "100vw", "60%" %}

宮崎の海岸沿いは高いところが多いですね。灯台の傍からの景色もとてもきれいでした。

{% image "./PXL_20260321_124202134.jpg", "灯台のそばからの景色", "100vw", "100%" %}

馬ヶ背という名前は盛り上がった岩肌が馬の背中に見えることから付いたみたいです。

上の写真の右側に見える柵の部分までいくことができて、そこからの景色もまた格別。

{% image "./PXL_20260321_130612786.MP.jpg", "馬ヶ背からの景色", "100vw", "100%" %}

駐車場に売店があって『へべすソフト』なるものを食べました (No Image)。

へべすというのは宮崎県日向市特産の香酸柑橘らしいです。

## 黒田の家臣・願いが叶うクルスの海
日向岬や馬ヶ背にほど近いところにある黒田の家臣と願いが叶うクルスの海にも行きました。

今更ながらマップを置いておきます。

{% image "./PXL_20260321_123144030.jpg", "日向岬周辺のマップ", "100vw", "100%" %}

黒田の家臣は潮が満ちていると渡れないところに歴史的な(?)墓があります。

アサリか何かがとれるらしく、潮干狩りをしている人がそれなりにいました。

下の画像は満潮時には水につかるようです。左右の見えないところで海に挟まれています。

{% image "./PXL_20260321_140711103_20260330232438.jpg", "黒田の家臣", "100vw", "100%" %}

黒田の家臣を離れたあと、願いが叶うクルスの海に行きました。面白い地名ですね。

下の写真の奥側から見ると、漢字の『叶』に見えることで有名なようです。

{% image "./PXL_20260321_142917895.jpg", "願いが叶うクルスの海", "100vw", "100%" %}

## 高千穂
日向岬を離れ、高千穂に向かいました。

天岩戸神社を軽く見て、いざ夕食です(昼は食べてません)。

高千穂牛！

{% image "./PXL_20260321_184114283.jpg", "高千穂牛", "100vw", "100%" %}

これまで食べた中で最も高級な肉だと思います。これはロースのステーキでしたが、柔らかくて非常においしかったです。

サーロインと赤身も食べさせてもらいましたが、どちらもおいしかったです(語彙力…)。

夕食をとった後は、高千穂峡のほうへ行きました。

{% image "./PXL_20260321_194935615.jpg", "高千穂峡", "100vw", "100%" %}

この期間はライトアップされていて、多くの人が訪れていました。きれいです。

なぜか月もありました。

{% image "./PXL_20260321_195234050.NIGHT.jpg", "月", "100vw", "100%" %}

実は高千穂周辺は『【推しの子】』の聖地になっていて、若干の聖地巡礼はできたかも？

機会があれば展望台とかも行ってみたいですね。

## 夜
入浴は日之影温泉駅でした。そこを通っていた鉄道が廃線になるまでは温泉のある駅として営業していたようです。

今は温泉施設として営業しています。どうやら敷地内に置かれた車両に泊まることができるらしい？

---

戻った後はお酒を飲みつつボドゲをしていました。

まずは『犯人は踊る』。サクッとできるのがよく、戦略性も運も必要なバランスの良いゲームだというのがメンバーの感想でした。

そして『最期のソワレ』。今流行りの(?)マーダーミステリーというやつです。前に同じメンバーで『ウェンディ、大人になって』をやりましたが、それと同じところが作っているものだそう。

アリバイについてほとんど話さなかったので普通に犯人に逃げ切られてしまいました。反省…。

マーダーミステリーも面白いのでいろいろ探してみたいですね。1回しか遊べないのが難点。

ボドゲを買ってみたいと思いつつ買ったことがないのでこの経験を参考に買ってみたい。

# 3日目
## 昼食
3日目はゆっくりスタート。昼食は『[味のおぐらチェーン](https://maps.app.goo.gl/1eCE7gwSrnB8WL7L7)』で『南ちゃんセット』というのを食べました。

この店の看板メニューのチキン南蛮とちゃんぽんが一緒になったセットです。

かなりのボリュームがあって、朝をあまり食べていなかったのに満腹になりました。

{% image "./PXL_20260322_121026287.jpg", "南ちゃんセット", "100vw", "100%" %}

## 鵜戸神宮 (うどじんぐう)
ここは海沿いにあって、本殿が洞窟の中にある神社です。

あまり写真を撮っていなかったのですが、こんな海の際にある神社を見るのは初めてな気がします。

{% image "./PXL_20260322_151428941_20260330223132.png", "鵜戸神宮", "100vw", "100%" %}

運玉というのを投げることができて、写真に写っている柵のすぐ近くの岩の上に縄が置かれていて、その中に入れることができると願いが叶うらしい。

男性は左手で投げるのですが、ちと難しい。1つくらいは入った気がします。

## 青島・青島神社
続いて青島に行きました。この頃から少し雨が強くなってきました。

青島に渡るのに橋があるのですが、遮るものがなく風が強かったので雨が横から叩きつけてきてズボンがめっちゃ濡れました。

青島には青島神社があります。島全体が境内になっているらしいです。

{% image "./PXL_20260322_162513604.jpg", "青島神社", "100vw", "100%" %}

Wikipediaによれば江戸時代までは禁足地だったようです。また、島全体が国の特別天然記念物に指定されているらしいです。

確かに、植生がジャングルっぽかったです。

さらに、島の周囲には『鬼の洗濯板』があります。岩が洗濯板のような形に削られています。

{% image "./PXL_20260322_163316300.jpg", "鬼の洗濯板", "100vw", "100%" %}

どうやってできたのか気になって調べてみたら、砂岩と泥岩が互いに層をなした地層が斜めに海上に露出したことで、侵食に弱い泥岩が大きく削られて今のような形になったようです。

自然はときに不思議な地形を作り出すものですね。

## 宮崎空港
長い旅路も終点が近づきました。まずは空港で夕食です。

いくつか店舗がありましたが、[回らない寿司屋](https://maps.app.goo.gl/6tHSHBGXVdT84riHA)を選択。

海鮮丼と太刀魚、レタス巻、ネギトロイクラ巻を食べました。ここまでちゃんとした回らない寿司に行ったのは初めてだと思います。

{% image "./PXL_20260322_181752163.jpg", "寿司", "100vw", "100%" %}

値段が書いてなかったり、メニューが羅列されていると何が何だか分からないですね。

いい経験ができたと思います。

その後はお土産を買って飛行機で無事に帰りました(飛行機が遅延して終電ギリギリだったのは別のお話)。

# おわりに
卒業旅行という名目で行った宮崎旅行でしたが、非常に良い旅だったと思います。

卒業おめでとう！

また、この旅に関わってくれた人にも感謝。
